# Snowflake Data Pipeline Project

## Overview

This project demonstrates an end-to-end data pipeline built using Snowflake. It follows a multi-layer architecture (Bronze, Silver, Gold) to transform raw data into business-ready insights.

## Tech Stack

* Snowflake
* SQL

## Architecture

* Bronze Layer: Raw data ingestion
* Silver Layer: Data cleaning and transformation
* Gold Layer: Aggregated data for analytics

## Key Features

* Data ingestion and staging
* Data transformation using SQL
* Analytical queries to generate insights

## Insights Generated

* Revenue trends
* Booking patterns
* Customer behavior analysis

## How to Run

1. Execute setup.sql
2. Run Bronze → Silver → Gold scripts
3. Execute analysis queries
